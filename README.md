# Awakened Steam Desktop Authenticator

Десктопный аутентификатор для Steam с расширенным функционалом управления аккаунтами.

## Возможности

- **Управление несколькими аккаунтами Steam** - добавляйте и управляйте множеством аккаунтов в одном приложении
- **Генерация 2FA кодов** - автоматическая генерация кодов двухфакторной аутентификации
- **Подтверждения** - просмотр и управление подтверждениями обменов и торговых предложений
- **Группировка аккаунтов** - организуйте аккаунты по группам для удобного управления
- **Автоматизация** - автоматическое принятие трейдов и управление торговой площадкой
- **Современный UI** - гибридный интерфейс на базе WinForms и WebView2

## Системные требования

- Windows 10/11
- .NET 8.0 Runtime
- WebView2 Runtime

## Установка

1. Скачайте последнюю версию из раздела Releases
2. Распакуйте архив
3. Запустите `SteamGuard.exe`

## Использование

### Добавление аккаунта

1. Нажмите кнопку "Добавить аккаунт"
2. Введите логин и пароль Steam
3. Следуйте инструкциям для подтверждения через email/Guard код
4. Аккаунт будет добавлен в список

### Управление подтверждениями

- Выберите аккаунт из списка
- Перейдите на вкладку "Подтверждения"
- Используйте кнопки "Принять" или "Отклонить" для управления подтверждениями
- Кнопка "Принять все" обработает все ожидающие подтверждения

### Настройки

В глобальных настройках доступны:
- Выбор группы по умолчанию для новых аккаунтов
- Автоматический ввод 2FA кодов
- Управление прокси-серверами

## Известные проблемы

⚠️ **Функционал прокси в данный момент не работает**

Система управления прокси находится в разработке. Вы можете добавлять прокси в настройках, но они не будут применяться к аккаунтам.

## Технологии

- C# / .NET 8.0
- WinForms
- WebView2
- Newtonsoft.Json
- SteamKit2

## Безопасность

- Все данные аккаунтов хранятся локально в зашифрованных `.mafile` файлах
- Приложение не отправляет данные на сторонние серверы
- Рекомендуется хранить резервные копии `.mafile` файлов и кодов восстановления

## Лицензия

Этот проект предназначен только для личного использования.

## Разработка

```bash
# Клонировать репозиторий
git clone https://github.com/Moorichoff/ASG.git

# Открыть проект
cd "Steam Guard"

# Собрать проект
dotnet build SteamGuard.csproj

# Запустить
dotnet run
```

## Структура проекта

- `MainForm.cs` - основная форма приложения
- `SteamAuth.cs` - работа с Steam аутентификацией
- `SteamServices.cs` - сервисы для работы с API Steam
- `AppSettings.cs` - управление настройками приложения
- `wwwroot/` - веб-интерфейс (HTML/CSS/JS)
