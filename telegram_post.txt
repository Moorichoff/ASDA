Awakened Steam Desktop Authenticator (ASDA)


—  Управление аккаунтами:
• Просмотр баланса кошелька Steam
• Управление подтверждениями трейдов/продаж
• Скрытие логинов (показ только первых и последних символов)
• Группировка аккаунтов для удобного управления
• Система избранных аккаунтов

—  Продвинутые функции:
• Поддержка прокси для каждого аккаунта
• Глобальный прокси для всех аккаунтов
• Экспорт данных аккаунтов в Excel

—  Open Source проект на GitHub
Активная разработка и регулярные обновления!
